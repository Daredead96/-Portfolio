# Game Store Analytics

## Overview / Обзор
Проект по аналитически-экономическому исследованию и предсказанию.

## Objectives / Цели
- Предсказания запасов месторождений нефти на основе линейной регресии
- Просчеты рисков и прибыли 
- Поиск слабых мест исходных условий

## Data Preparation / Подготовка данных
**Data Sources / Источники данных:**
- Датасеты с даными по трем регионам (есть синтетические данные)

**Preprocessing / Предобработка:**

- Анализ распределений , аномалий, корреляций.

## Key Findings / Ключевые выводы

### Distribution / Распределение
- Все признаки кроме случая воторого региона, имеют нормальное распредление с выбросами (пиками)
- Второй регион имеет распределение похожее на набор дискретных значений.
### ML / модели машинного обучения 
- Для всех троех регионов обучены модели, с приемлимой метрикой RMSE < std
- Второй регион в следсвии необычного распределения пресдказан лучше всего.
- Получены предсказания по продукту на каждый регион.

### Аnalyze / Анализ
- Все регионы не дотягивают до безубыточности по среднему содержанию нефти на одноместорождение (из 200-т)
- Самым экономически выгодным регоном является 2й

## Saport_idea / Вспомогательные идеи (анализ слабых мест проекта)
- Выдвинуты идеи как улучшить задачу для большей продуктивности
- Преложен вариант по повышению качества оценки скважин.

## Conclusion / Заключение
Проект успешно достиг поставленных целей, моедли для трех регионов обучены, продукт предсказан и благодаря этому просчитаны риски. Выбран лучший с точки зрения прибыли регион. Даны рекомендации и идеи как улучшить подход  и условия к задаче для улучшения  резуьтатов.
