# Game Store Analytics

## Overview / Обзор
Проект по аналитическому исследованию данных для магазина видеоигр с целью выявления закономерностей и формирования стратегических рекомендаций.

## Objectives / Цели
- Анализ исторических данных о продажах видеоигр
- Выявление определяющих успешность игры факторов
- Исследование региональных особенностей игровых предпочтений
- Формирование портретов пользователей по ключевым регионам
- Проверка статистических гипотез о пользовательских рейтингах

## Data Preparation / Подготовка данных
**Data Sources / Источники данных:**
- Датасет с историческими данными о продажах видеоигр

**Preprocessing / Предобработка:**
- Приведение названий столбцов к нормализованному виду
- Обработка пропусков в критических полях (названия, годы выпуска)
- Сохранение пропусков в оценках пользователей и критиков для дальнейшего анализа
- Коррекция типов данных (год выпуска, оценки)
- Добавление столбца с общими продажами
- Анализ распределений и аномалий

## Key Findings / Ключевые выводы

### Historical Dynamics / Историческая динамика
- До 1990 года: редкие релизы (становление индустрии)
- 1990-2010: активный рост количества выпускаемых игр
- После 2010: спад, связанный с усложнением разработки

### Platform Analysis / Платформенный анализ
- Лидеры по продажам: PS2, X360, PS3, Wii, DS, PS
- Жизненный цикл платформ: ~10 лет, пиковая активность - 5 лет
- Текущие тренды: PS4 и XOne демонстрируют успех, но идут на спад

### Ratings Influence / Влияние оценок
- Продажи слабо зависят от оценок пользователей
- Оценки критиков оказывают значимое влияние на коммерческий успех
- Умеренная положительная корреляция между оценками критиков и пользователей

### Genre Preferences / Жанровые предпочтения
- Наибольший спрос: Action, Shooter, Sports
- Перспективные жанры: Role-Playing
- Стабильный интерес: Fighting, Platform

### Regional Features / Региональные особенности
- Северная Америка и Европа: PS4 (топ-1), жанры - Shooter/Action, рейтинг "M"
- Япония: 3DS (топ-1), жанр - Role-Playing, неопределенный рейтинг
- Европа: уникальное присутствие PC-гейминга в топе платформ

## Business Recommendations / Бизнес-рекомендации

### Strategic / Стратегические
- Фокус на актуальные платформы - использовать данные за 2-3 года для прогнозирования
- Жанровая стратегия - усилить позиции в Action, Shooter, Sports, развивать Role-Playing
- Региональная диверсификация - адаптировать ассортимент под локальные предпочтения

### Tactical / Тактические
- Мобильный гейминг - провести отдельное исследование перспектив направления
- Киберспорт - мониторить развитие для своевременного входа на рынок
- Качество против количества - учитывать возросшую сложность разработки при планировании релизов

### Operational / Операционные
- Данные о рейтингах - дополнить информацию сертификациями PEGI/IARC
- Маркетинговые кампании - делать акцент на оценках критиков
- Ассортиментная политика - учитывать региональные портреты пользователей

## Next Steps / Дальнейшие шаги
- Провести анализ мобильного гейминга как перспективного направления
- Углубить исследование киберспортивного сегмента
- Реализовать систему динамического обновления аналитических периодов
- Настроить регулярный мониторинг изменений в игровых предпочтениях

## Conclusion / Заключение
Проект успешно достиг поставленных целей, выявив ключевые закономерности в игровой индустрии и сформировав практические рекомендации для стратегического планирования. Полученные инсайты позволяют оптимизировать бизнес-процессы и адаптироваться к быстро меняющемуся рынку видеоигр.
